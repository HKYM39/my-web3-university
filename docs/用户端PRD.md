# **学习者端（Learner App）PRD**

版本：v1.0  
适用对象：前端团队、后端团队、产品团队、QA  
范围：Web3 大学平台学习者可访问的所有功能。

---

# **1. 产品目标（Learner）**

为学习者提供基于 Web3 的课程学习体验，包括：

- 链上登录（钱包即账号）

- 课程浏览、购买、学习

- NFT 课程访问权限

- 用户中心（签名资料修改）

- 学习挖矿奖励

目标是：使学习者在链上获得可验证的学习凭证，并通过代币激励驱动持续学习。

---

# **2. 用户范围（Persona）**

| 用户类型     | 行为                  | 动机                      |
| -------- | ------------------- | ----------------------- |
| Web3 学习者 | 浏览课程、购买课程、学习课程、获得奖励 | 获得技能、学习挖矿奖励、拥有不可篡改的学习凭证 |
| Web3 初学者 | 钱包登录入门用户            | 尝试 Web3 学习体验            |
| 高价值学习者   | 付费购买课程              | 获取专业内容、享受链上资产稳定性        |

---

# **3. 功能需求（FR）**

---

## **3.1 Web3 登录模块**

| 编号       | 功能       | 描述                          | 备注       |
| -------- | -------- | --------------------------- | -------- |
| L-FR-001 | 钱包连接     | 支持 MetaMask / WalletConnect | 核心入口     |
| L-FR-002 | 签名登录     | 使用 nonce 机制防重放              | 安全性要求    |
| L-FR-003 | 获取用户链上资产 | YIDENG 余额、NFT               | 调用链上 RPC |

---

## **3.2 课程浏览模块**

| 编号       | 功能       | 描述                 |
| -------- | -------- | ------------------ |
| L-FR-010 | 课程列表页    | 浏览所有课程、价格、简介       |
| L-FR-011 | 课程详情页    | 包含课程简介、目录、价格、创作者信息 |
| L-FR-012 | NFT 权益判断 | 若已购买则自动展示内容        |

---

## **3.3 课程购买模块**

### 功能需求

| 编号       | 功能       | 描述                  |
| -------- | -------- | ------------------- |
| L-FR-020 | 授权代币     | 调用 approve 允许课程合约扣费 |
| L-FR-021 | 购买课程     | buyCourse(courseId) |
| L-FR-022 | 接收课程 NFT | SBT，不可转让            |
| L-FR-023 | 购买校验     | 防止重复购买              |

### 流程（学习者视角）

```
课程详情页 → 点击购买 → 授权 approve → 链上交易 → 成功 → NFT 发放 → 解锁课程内容
```

| L-FR-032 | 播放进度记录 | 用于学习挖矿         |

---

## **3.5 学习挖矿模块（Learning Mining）**

| 编号       | 功能     | 描述                 |
| -------- | ------ | ------------------ |
| L-FR-040 | 学习进度上报 | 前端定期上报时间切片         |
| L-FR-041 | 完成课程奖励 | 完播 ≥ 80% 即可领取      |
| L-FR-042 | 每日学习签到 | 每日可领取基础奖励          |
| L-FR-043 | 领取奖励   | 调用 rewardPool 奖励用户 |

奖励类型：YIDENG Token

# **4. 非功能性需求（NFR）**

| 项目  | 指标             | 描述          |
| --- | -------------- | ----------- |
| 性能  | 白屏 < 2s        | Next.js SSR |
| 可用性 | ≥ 99.5%        | 核心服务稳定      |
| 安全性 | 签名防重放、NFT 不可转让 | 用户资产保护      |
